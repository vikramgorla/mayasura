## Key Architectural Decisions

1. **SQLite + Drizzle ORM** — Single-file database, no external DB dependency, easy self-hosting
2. **Next.js App Router** — Server Components by default, client components only where needed
3. **CSS Custom Properties** — Design system tokens in CSS, consumed by both platform UI and consumer sites
4. **Template-as-data** — Templates are TypeScript objects with design tokens, not separate codebases
5. **AI graceful degradation** — All AI features work without API key (show prompts/placeholders)
6. **Standalone output** — Docker-friendly, no node_modules in production image
7. **JWT auth** — Simple, stateless, no external auth provider needed
8. **Zinc-based dark mode** — Warmer than slate, consistent across all surfaces
9. **Framer Motion** — Scroll-triggered animations on landing page, wizard step transitions
10. **No fake data** — All numbers on landing page are real from the codebase (Constitution Principle II)
